# React Introduction:
- React is build on top of JavaScript.
- React is Created & maintained by facebook/meta.
- React is javascript library for web & native user interfaces.
- React is SPA.(Single Page Application)
Eg., Netflix is created using React.

- Components:
    - Class based components
    - Function based components

- JSX:  JavaScript + XML


# Command to create react app
    - npx create-react-app <App-Name>
    - cd <App-Name>
    - npm start (App will launch in localhost:3000)


> How to use Components in index.js
1. <App/>
2. <App><App/>

> There are 4 important files in React Application.
    - src/index.js
    - src/index.css
    - src/App.js
    - src/App.css
